## 制作随身容器的方法

### 1. 注册容器组件
首先需要在 `modmain.lua` 中注册你的容器组件：

```lua
-- 注册容器组件
AddComponent("your_container_component")
```

### 2. 创建容器组件
创建一个组件文件（如 `your_container_component.lua`），参考云青松容器的实现：

```lua
-- components/your_container_component.lua
local YourContainerComponent = Class(function(self, inst)
    self.inst = inst
    self.boxkey = nil  -- 容器标识
end)

-- 初始化容器
function YourContainerComponent:OnLoad()
    -- 创建容器实例
    local container = self.inst:GetPocketDimensionContainer("your_box_key")
    if container == nil then
        container = SpawnPrefab("pocket_dimension")
        container.entity:SetParent(self.inst.entity)
        self.inst:SetPocketDimensionContainer("your_box_key", container)
    end
end

-- 打开容器
function YourContainerComponent:OpenContainer(doer)
    if doer ~= nil and doer.components.playercontroller ~= nil then
        doer.components.playercontroller:OpenContainer(self.inst)
    end
end

return YourContainerComponent
```

### 3. 创建预制物
创建一个可以携带的容器物品（来源4：棱镜/scripts/prefabs/worldbox_legion.lua）：

```lua
-- prefabs/your_portable_container.lua
local assets = {
    Asset("ANIM", "anim/your_container.zip"),  -- 你的容器动画
    Asset("ATLAS", "images/inventoryimages/your_container.xml"),
}

local function fn()
    local inst = CreateEntity()
    
    -- 添加基本组件
    inst.entity:AddTransform()
    inst.entity:AddAnimState()
    inst.entity:AddNetwork()
    
    -- 添加容器组件
    inst:AddComponent("your_container_component")
    
    -- 添加可携带组件
    inst:AddComponent("inventoryitem")
    inst.components.inventoryitem:SetOnDroppedFn(function(inst)
        -- 掉落时关闭容器
    end)
    
    -- 添加可装备组件（可选，如果想让容器可以装备在手上）
    inst:AddComponent("equippable")
    
    -- 添加交互组件
    inst:AddComponent("inspectable")
    
    return inst
end

return Prefab("your_portable_container", fn, assets)
```

### 4. 注册预制物
在 `modmain.lua` 中注册：

```lua
PrefabFiles = {
    "your_portable_container",
}
```

### 5. 添加制作配方（可选）
参考来源1的配方格式（来源1：棱镜/scripts/recipes_legion.lua）：

```lua
AddRecipe("your_portable_container", {
    Ingredient("cutstone", 2),
```

## 关键实现要点

1. **使用 `PocketDimensionContainer`**：这是DST内置的随身容器系统，通过 `inst:GetPocketDimensionContainer(key)` 和 `inst:SetPocketDimensionContainer(key, container)` 来管理

2. **物品转移逻辑**：参考来源2的代码，当需要转移物品时，使用 `container.components.container:RemoveAllItems()` 取出所有物品

3. **容器标识**：每个容器需要一个唯一的 `boxkey` 来区分不同的容器实例

（以下为补充知识）

知识库没有覆盖的部分：
- **容器UI定制**：如果需要自定义容器界面，需要额外添加 `widget` 组件
- **容量限制**：默认容器容量为12格，可以通过修改 `container.components.container:SetNumSlots(n)` 来调整
- **保存/加载**：需要实现 `OnSave` 和 `OnLoad` 函数来保存容器内容，否则容器物品会在存档重载时丢失

这些内容在标准DST API文档中有详细说明，但知识库中未收录相关文档。

根据知识库检索结果，DST中食物的腐败速度主要通过 `perishable` 组件进行修改。以下是具体方法：

## 核心修改方式

使用 `SetPerishTime()` 方法设置腐败时间（单位：秒），然后调用 `StartPerishing()` 启动腐败进程。

### 示例代码

```lua
-- 添加腐败组件
inst:AddComponent("perishable")

-- 设置腐败时间（使用TUNING常量）
inst.components.perishable:SetPerishTime(TUNING.PERISH_SUPERSLOW)  -- 约40天

-- 启动腐败
inst.components.perishable:StartPerishing()

-- 设置腐败后替换的物品
inst.components.perishable.onperishreplacement = "spoiled_food"
```

## 可用的TUNING常量（来源2：棱镜/scripts/preparedfoods_legion.lua）

| 常量名 | 对应天数 | 说明 |
|--------|---------|------|
| `TUNING.PERISH_ONE_DAY` | 1天 | 最快腐败 |
| `TUNING.PERISH_TWO_DAY` | 2天 | 快速腐败 |
| `TUNING.PERISH_SUPERFAST` | 3天 | 超级快 |
| `TUNING.PERISH_FAST` | 6天 | 快 |
| `TUNING.PERISH_FASTISH` | 8天 | 较快 |
| `TUNING.PERISH_MED` | 10天 | 中等 |
| `TUNING.PERISH_SLOW` | 15天 | 慢 |
| `TUNING.PERISH_PRESERVED` | 20天 | 较慢 |
| `TUNING.PERISH_SUPERSLOW` | 40天 | 最慢 |

## 自定义腐败时间

也可以直接传入自定义秒数：

```lua
-- 设置100天腐败（1天 = 480秒）
inst.components.perishable:SetPerishTime(480 * 100)
```

## 注意事项

1. **必须调用 `StartPerishing()`** 才能让腐败开始计时（来源1：能力勋章/scripts/prefabs/mandrakeberry.lua）
2. **`onperishreplacement`** 可设置腐败后变成什么物品，常见设置为 `"spoiled_food"`（腐烂食物）
3. 如果食物本身有保质期，修改 `perishable` 组件即可覆盖默认值
